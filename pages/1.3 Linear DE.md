# Type 1
	- ## Base Equation:
	  \begin{equation*}
	  \{
	  \begin{array}{}
	      a_i = ra_{i-1}\text{, }       i = 1,2,3 \\
	      a_0
	  \end{array}
	  \end{equation*}
	- ### Edge Cases
		- #### -1 < r < 1
		  $a_n = 0$
		- #### R = 1
		  $a_n = a_0$
		- #### r < -1, |r| > 1
		  $a_i > 0$ if i is even
		  $a_i < 0$ if i is odd
		- #### r = -1
		  $a_i = \pm a_0$
- # Type 2
  A number a is an equilibrium of a DE if $$a_i = a$$
	- ## Base Equation:
	  \begin{equation*}
	  \{
	  \begin{array}{}
	    a_i = f(f_{a-1}) \text{, if } a_i = a\\
	    a_0
	  \end{array}
	  \end{equation*}
