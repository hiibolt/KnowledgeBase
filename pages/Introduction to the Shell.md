## Terminology
	- ## `stdin`
	  Standard input stream
	- ## `stdout`
	  Standard output stream
	- ## `stderr`
	  Standard error stream
- ## Simple Commands
	- ### `ls`
	  Lists the contents of the current directory
	- ### `pwd`
	  Prints the path of the current working directory
	- ### `clear`
	  Clears the shell screen
	- ### `date`
